Rappels. Les anciens certificats « C-2026-NNN » ne sont pas renumérotés. Le séquentiel par artiste est stable (réservé à la création, comme les annexes). Le n° Sage étant requis, le numéro est complet dès la production. Le numéro ne contient pas l'année ; le nom de fichier porte la date (n° d'inventaire + date), le numéro complet restant dans le PDF. Un éventuel doublon le même jour reçoit un suffixe « (2) ».
