Fiche d'œuvre — section Description

Le bouton doré « Générer la description » envoie la photo de l'œuvre + les consignes (galerie + artiste) à Claude et remplit le champ. « Copier pour ChatGPT » reste comme repli (copier-coller, hors ligne).

Seules les données de l'œuvre (image + caractéristiques) sont envoyées — jamais de données client. Relis et ajuste avant d'enregistrer.

Réglages → IA

La clé est saisie une seule fois, chiffrée dans le coffre de Windows (jamais affichée ni stockée en clair).

Assistant de description (Claude)

Optionnel — sans clé, l'app fonctionne normalement (le bouton renvoie ici).

Clé API Anthropic

Clé définie · chiffrée dans le coffre Windows

À créer sur console.anthropic.com (payant à l'usage, ~0,4 ¢ par description). Distinct d'un abonnement Claude/ChatGPT.

Consignes générales de la galerie

S'ajoute aux consignes propres à chaque artiste (champ « Aide à la description IA » sur la fiche artiste).

Clé API non configurée

Pour générer une description automatiquement, ajoute ta clé Anthropic dans Réglages → IA. La fonction est optionnelle ; tu peux toujours utiliser « Copier pour ChatGPT ».
